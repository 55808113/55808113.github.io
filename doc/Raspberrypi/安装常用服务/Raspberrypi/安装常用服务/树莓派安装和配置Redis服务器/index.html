树莓派安装和配置Redis服务器

在使用nodejs的时候，使用到了Redis服务。所以想在树莓派上进行安装。

树莓派安装Redis服务器

我不喜欢敲命令就直接界面操作了，到自己所在的目录里找到下载的文件redis-6.0.1.tar.gz

把这个文件解压、剪切放到/usr/local/lib目录下面。

然后启动命令行窗口，目录切换到/usr/local/lib/redis-6.0.1执行make


开始编译程序。

树莓派配置Redis服务器

使用redis客户端本地用127.0.0.1访问正常，远程ip访问就会报错，解决方法，修改redis-6.0.1/redis.conf文件。然后启动Redis的时候需要后面添加上配置文件的路径。

修改服务地址

为了让远程能够访问到Redis服务器需要修改一下IP地址。打开redis-6.0.1/redis.conf文件。找到如下图，把本机的网络地址添加到后面址bind 192.168.0.202 127.0.0.1

关闭Redis服务保护模式

Redis服务处于保护模式，我们需要修改配置文件redis-6.0.1/redis.conf。将NETWORK下的protected-mode yes修改为protected-mode no，然后启动Redis服务src/redis-server ./redis.conf
