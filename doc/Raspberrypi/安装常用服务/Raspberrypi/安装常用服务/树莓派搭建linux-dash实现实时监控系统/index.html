树莓派搭建linux-dash实现实时监控系统

说明

linux-dash支持很多的平台。

  • Node.js (recommended)
  • Go
  • Python
  • PHP

这里我只介绍一下我了解的Node.js平台

下载linux-dash

https://github.com/afaqurk/linux-dash

安装

把下载的压缩文件解压到/usr/local/src/linux-dash-master目录中。

执行下面的语句。来更新一下支持的类包。这里的参数--production表示，只安装项目所使用的类包。

npm install 安装所有依赖
npm install –production 安装生产依赖
npm install xx –save 安装XX到生产环境依赖
npm install –save-dev 安装XX到开发环境依赖

修改配置

主要需要修改的是IP地址和端口号。
默认的端口号是80.
