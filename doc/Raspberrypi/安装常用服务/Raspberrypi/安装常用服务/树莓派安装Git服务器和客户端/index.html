使用gitblit管理项目

登陆系统

启动 gitblit 界面，在左上角有个登录框，初始值用户名和密码均为 admin，然后便可以添加用户和团队并设置其权限。

设置界面为中文

使用admin登陆系统以后，点击右上角的用户图标，弹出菜单中选择users

出来如下界面，然后选择点击admin。

会弹出如下界面，选择为简体中文就行了，点击保存。

添加版本管理的用户

在这个界面下，选择添加用户

在如下界面中，设置相关信息保存就行了。

创建版本库

在如下界面中选择创建版本库

在弹出的界面中根据自己的库的特点选择访问策略就行了

管理用户访问版本库的权限

点击版本库中的编辑，在弹出的界面中设置那一些用户可以访问就行了。
