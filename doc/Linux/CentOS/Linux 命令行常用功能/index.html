Shell小窍门

  1. 命令补齐
    比如”dat”，按Tab键，Linux会把它补充成为”date”
  2. 文件名补齐
    当你输入到ls a.t的时候，按Tab键，Shell会帮你补齐该文件名，即ls a.txt。
  3. 历史命令
    在Shell中，可以用向上箭头，或history命令来查看之前输入的命令。
  4. 中止与暂停命令
    当一个命令运行时，如果中途想要停止它，那么可以用快捷键Ctrl+C。如果只是想暂时停止，那么可以使用快捷键Ctrl+Z。

文件相关命令

目录处理命令

tree

树形结构显示目录，需要安装tree包

mkdir

(make directories)创建目录

  • -p递归创建目录，若无父目录，则创建p(parent)

rmdir

(remove empty directories)删除空目录

pwd

(print working directories)显示当前目录

cd

(change directory)这是一个非常基本，也是大家经常用于切换文件目录的命令。

ls

显示文件或目录，查看文件与目录的命令，list的缩写，它的参数非常多。

  • -l列出文件详细信息l(list)
  • -h列出文件详细信息的时候大小为k，m等信息就不是字节的单位
  • -a列出当前目录下所有文件及目录，包括隐藏的a(all)
  • -d 列出当前目录信息

cp

(copy)复制文件或目录

  • -r复制目录
  • -p保留文件属性。例如：修改时间和创建时间都是一样的。

mv

(move)移动或重命名

rm

删除文件(

  • -r 递归删除，可删除子目录及文件，
  • -f 强制删除

文件处理命令

touch

创建空文件

echo

创建带有内容的文件。

cat

查看文件内容(适合文件比较少的文件)

  • -n显示行号

tac

倒着查看文件内容与cat正好相反(适合文件比较少的文件)

more

分页显示文本文件内容，但不能向上翻页，有时候在显示长文本的时候使用这个还是非常方便的。

  • (空格)或者f翻页
  • (Enter)换行
  • q或者Q退出

less

分页显示文本文件内容，可以向上翻页，有时候在显示长文本的时候使用这个还是非常方便的。

  • pageup往上翻页
  • pagedown往下翻页
  • 上剪头上一行
  • 下剪头下一行

head

显示文件前几行

  • -n指定行数不写默认为10行

tail

显示文件最后几行，在实时显示日志信息的时候这个语句非常的好用。

  • -n指定行数
  • -f动态显示文件末尾内容

wc

统计文本中行数、字数、字符数

grep

在文本文件中查找某个字符串

find

在文件系统中搜索某文件

连接命令

ln

(link)创建一个文件的快捷方式。

  • -s创建软连接。

快捷方式：软连接和硬连接
软连接：像window的快捷方式
硬连接：就是把文件索引复制一个。原文件的删除了，只是删除了原文件的索引。硬连接的文件索引还是存在的。

权限管理

chmod

(change mode)使用权限 : 所有使用者

  • -R递归修改

三种基本权限

  • r 读(read) 数值表示为4
  • w 写 (wirte) 数值表示为2
  • x 可执行(executable) 数值表示为1

对于文件。

  • r:读
  • w写
  • x可执行

对于目录。

  • r:可以ls查看
  • w:添加或者删除目录里的文件
  • x:可以cd到目录里
    所以对于目录如果有r就一定有x权限

-rw-rw-r--、drwxr-xr-x一共十个字符，分成四段。
第一个字符

  • “-”表示普通文件；
  • “l”软链接；
  • “d”表示目录；

第二三四个字符
“rw-”表示当前所属用户的权限。所以用数值表示为4+2=6
第五六七个字符
“rw-”表示当前所属组的权限。所以用数值表示为4+2=6
第八九十个字符
“r–”表示其他用户权限。所以用数值表示为4
所以操作此文件的权限用数值表示为664

chown

(change ownerp)命令来改变文件的拥有者和用户组，只有root才能修改所有者

  • -R递归修改

例如：还以jdk-7u21-linux-i586.tar.gz为例。属于用户hadoop，组hadoop
要想切换此文件所属的用户及组。可以使用命令。

chgrp

(change group)改变文件或目录的所属组

用户及用户组管理

/etc/passwd

存储用户账号(任何用户都能查看)


说明

序号 说明
0 用户名称
1 密码占位符
2 用户UID
3 组GID
4 注释信息
5 用户家目录
6 linux命令解释器（普通用户：/bin/bash;伪用户：/sbin/nologin）
用户UID 系统中约定的含义
0 超级管理员，最高权限，有着极强的破坏能力
1~200 系统用户，用来运行系统自带的进程，默认已创建
201~999 系统用户，用来运行用户安装的程序，所以此类用户无需登录系统
1000+ 普通用户，正常可以登陆系统的用户，权限比较小，能执行的任务有限

/etc/group

存储组账号

序号 说明
0 组名称
1 密码占位符
2 组GID

/etc/shadow

存储用户账号的密码(root才能查看)

序号 说明
1 用户名
2 密码为一长串字符。!!为空密码
3 最近一次变更密码，从1970年到现在过了多少天
4 密码最少使用天数，0无限制
5 密码最长使用天数，99999不过期
6 密码到期前，密码到期前，系统会在密码到期前7天提醒变更密码
7 密码到期后，密码到期后2天强制提示变更用户密码
8 账户失效时间，从1970年起，账户在这个时间前可用，到期后失效

/etc/gshadow

存储用户组账号的密码(不怎么常用)

useradd

添加用户 useradd 用户名

  • -u<uid> 　指定用户ID。
  • -g<群组> 　指定用户所属的群组。
  • -G<群组> 　指定用户所属的附加群组。

添加用户默念缺省文件


添加用户的默认密码设置文件

userdel

删除用户userdel 用户名

  • -r 把用户的/home/用户名家目录一起删除

adduser

添加用户adduser 用户名

groupadd

添加用户组groupadd 组名

groupdel

删除用户组groupdel 组名

passwd

修改密码passwd root 给root设置密码
su root
su – root

/etc/profile

系统环境变量

bash_profile

用户环境变量

.bashrc

用户环境变量

su user

切换用户，加载配置文件.bashrc

  • -c只执行一次命令而不切换用户身份
  • -切换用户，加载配置文件/etc/profile ，加载bash_profile，更改文件的用户及用户组

使用上面的命令可以直接查看cd的帮助信息。

因为1p章节是bash的帮助信息，所以在使用”man 1p cd“后，通过在man中输入”/cd”查找相关的信息才能看到cd的帮助信息。
