4.安装完成后的优化工作

4.1.更换软件源

System Settings >>> Software&Updates

4.2如果是虚拟机安装VMware tools

  • 在VMware菜单中虚拟机>安装VMware tools.
  • 打开光盘，把里面的文件全选，复制粘贴到download的下载目录下
  • 命令sudo ./vmware-install.pl加图一的英文按回车，并输入开机的管理员密码。接着就是选择yes回车一步一步就可以了。

5.Ubuntu美化

5.1安装美化工具

安装完成后我们打开Tweaks

5.2安装美化工具扩展gnome-shell-extensions

需要重新启动一下电脑才能在扩展中看到

将“User themes”设置为ON

5.3安装主题

在官网https://www.pling.com/s/Gnome/browse的左边有一个GTK3 Themes，选择一款自己喜欢的。

  • 点击Files找一个下载
  • 下载完成后我们解压
  • 再将文件夹移动到/usr/share/themes/

然后去Tweaks里更换主题–>应用程序即可

5.4下载Dash to Dock

进入 GNOME Shell Extensions 官网，第一次打开会有提示，点击带下划线的链接进行安装。下次直接点击浏览器工具栏上的图标即可访问该网站。

插件地址：https://extensions.gnome.org/extension/307/dash-to-dock/

解压重命名文件夹为dash-to-dock@micxgx.gmail.com，然后把文件复制到home文件下的.local/share/gnome-shell/extensions/

需要重新启动一下电脑才能看到
